**dupes** - утилита, предназначенная для очистки директорий от дубликатов.

python -m venv .venv && .venv\Scripts\activate 

**Запуск программы: ** (внутри клонированного проекта) python main.py --path "Путь, необходимой директории".

**Доступные параметры при запуске:**
- --path: целевая папка для сканирования на дубликаты
- --action: Принимает следующие параметры:
      dry-run - сканирование с выводом файлов-дубликатов
      move    - dry-run + перемещение дубликатов в папку 'for del'
      del     - dry-run + move + удалние файлов из папки 'for del'
